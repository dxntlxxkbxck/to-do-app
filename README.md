Описание проекта
Суть: Создать список задач, куда можно добавлять пункты. У каждой задачи есть кнопка «Удалить» и кнопка «Выполнено» (меняет стиль текста).
Технические требования (все выполнены):
	•	Vite — современный инструмент сборки
	•	Данные хранятся в массиве объектов  tasks = { id: 1, text: 'Купить хлеб', completed: false } 
	•	Функция  render()  — полностью очищает контейнер в DOM и отрисовывает список заново
	•	Добавление новой задачи через input и кнопку (или Enter)
	•	Удаление задачи по ID (используя  splice() )
	•	Мини-вызов: Счетчик «Всего задач: X, Выполнено: Y» — обновляется автоматически
Быстрый старт

# 1. Создай проект
npm create vite@latest todo-app -- --template vanilla
cd todo-app
npm install

# 2. Замени файлы:
#   - index.html (корень)
#   - main.js (корень) 
#   - src/style.css
#   - README.md (этот файл)

# 3. Запуск
npm run dev
